## Installing Charts from this Repository

Add the Repository to Helm:

```sh
helm repo add rafaelcalleja-helm-charts https://rafaelcalleja.github.io/helm-charts
```

Install a chart:

```sh
helm install rafaelcalleja-helm-charts/[CHART_NAME]
```
